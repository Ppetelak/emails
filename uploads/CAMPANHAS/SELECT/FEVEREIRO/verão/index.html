Combate contra a leucemia

Os exames de rotina são indispensáveis para a prevenção da doença, que afeta as células sanguíneas.

Não existe uma regra para prevenir a leucemia, mas com alguns hábitos é possível diminuir os riscos de desenvolver a doença:

Testes genéticos

Evitar o tabagismo

Possuir uma alimentação equilibrada

Evitar a exposição à substâncias tóxicas

Sua atitude pode fazer a

diferença. Apoie essa luta!
